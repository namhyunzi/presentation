안전한 개인정보 전달 체계

쇼핑몰이 JWT와 함께 주문정보를 택배사에 전달하여 직접적인 개인정보 전송을 방지

JWT 검증 및 권한 확인

SSDM은 JWT 토큰과 사전 약속된 요청 필드를 검증하여 보안성 강화

보안뷰어를 통한 제한적 접근

필요한 정보만 보안뷰어에 표시하여 택배사에 제한된 접근 권한 부여

세션 ID 관리

보안뷰어는 세션 ID로 관리되며 최대 2번까지만 요청 가능

보안뷰어는 기본 12시간, 최대 2번까지 연장 가능하며, JWT는 15분 후 만료됩니다. 만료된 JWT로는 보안뷰어를 다시 요청할 수 없습니다.
