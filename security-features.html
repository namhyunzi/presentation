JWT 토큰 보안 관리

15분 후 자동 만료되는 JWT 토큰으로 불법 접근 차단

접근 권한이 제한된 토큰 발급으로 필요한 정보만 제공

보안뷰어 시스템

최대 2회까지만 요청 가능한 제한적 접근 시스템

기본 12시간, 최대 2번까지 연장 가능한 유효기간

보안뷰어 세션 ID로 접근 기록 및 관리

만료 시 텍스트 표시 및 접근 차단

보안뷰어 보호 메커니즘

JWT가 만료되면 다시 접근할 수 없으며, 만료된 페이지는 개인정보를 표시하지 않습니다. 이를 통해 개인정보가 권한 없이 노출되는 것을 원천적으로 차단합니다.
